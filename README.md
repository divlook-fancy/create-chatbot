# Create Chatbot

간단한 챗봇을 만들 수 있습니다.
현재 [카카오톡 플러스친구](https://github.com/plusfriend/auto_reply)만 지원합니다.

## Run the app

```bash
// install
$ npm i

// sqlite3
$ npm i sqlite3 --no-save

// start
$ DEBUG=create-chatbot:* npm start
```
